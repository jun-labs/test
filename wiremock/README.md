# [Wiremock](https://github.com/wiremock/wiremock)



<br/><br/><br/><br/>

## Getting Started

> You should install jdk 17 or higher. <br/>

<br/><br/><br/>

## Run Test

````text
$ ./gradlew test
````

<br/><br/>

## Run Build

````text
$ ./gradlew build
````

<br/><br/>


## Env

&nbsp;&nbsp; - Java 17  <br/>
&nbsp;&nbsp; - SpringBoot 3.0.0 <br/>
&nbsp;&nbsp; - Wiremock 4.0.1 <br/>
